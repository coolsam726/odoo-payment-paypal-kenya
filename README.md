# PayPal - Kenya
This addon extends the Odoo's PayPal Payment provider to support Kenyan Currency (KES) and auto-convert into USD during transactions.

## Features
- Supports Kenyan Shilling (KES) for Paypal transactions.
- Automatic conversion from KES to USD during payment processing.

## Installation
1. Place the `payment_paypal_kenya` folder in your Odoo addons directory.
2. Restart your Odoo server.
3. Activate the developer mode in Odoo.
4. Go to Apps, search for "PayPal - Kenya" and install the module.
5. Configure your PayPal payment acquirer to use the new KES currency option.
6. Ensure you have a valid PayPal account set up for processing payments.
7. Test the payment process to confirm that KES transactions are being converted to USD correctly.
8. Enjoy seamless Paypal payments in Kenyan Shillings!

## Compatibility
This addon is compatible with Odoo version 19.0 and above.
